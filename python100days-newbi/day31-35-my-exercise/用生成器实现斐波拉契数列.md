使用生成器（Generator）实现斐波那契数列的核心优势是**节省内存**，因为它按需即时生成数值，而不是一次性将整个数列加载到内存中。 [[1](https://blog.csdn.net/weixin_65188536/article/details/134766914), [2](https://www.cnblogs.com/BigJ/p/7397729.html)]

以下是使用 Python 实现的经典示例： [[1](https://zhuanlan.zhihu.com/p/620103258)]

python

```
def fibonacci(n):
    """生成前 n 个斐波那契数的生成器"""
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

# 示例：输出前 10 个数
for num in fibonacci(10):
    print(num)
```

请谨慎使用此类代码。

无限斐波那契数列

如果数列长度未知，可以使用 `while True` 创建一个无限生成器： [[1](https://zhuanlan.zhihu.com/p/620103258)]

python

```
def fibonacci_infinite():
    """无限生成斐波那契数的生成器"""
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

# 示例：获取前 10 个数
import itertools
fib_gen = fibonacci_infinite()
first_10 = list(itertools.islice(fib_gen, 10))
print(first_10)
```

请谨慎使用此类代码。

**yield 的工作原理**：
每次调用 `yield` 时，生成器会返回当前的值并**暂停**执行；当下一次调用（如通过 `next()`）时，生成器会从上次暂停的位置继续执行，从而实现动态推导。 [[1](https://www.cnblogs.com/yanpeng1535/p/18529883)]
